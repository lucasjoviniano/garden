Digital Garden

Introdução

Essa é minha Wikipédia pessoal, onde eu tento postar tudo que sei.

Esse tipo de blog é chamado de Digital Gardens. Existe uma que discute a criação de wikis do tipo.

Porque fazer isso?

Já tentei o estilo de blog tradicional, onde tudo funciona com um botão de publicar. Infelizmente, não funcionou pra mim.

Apesar disso, gosto de fazer anotações de quse tudo, pois me ajudam a entender os mais diversos assuntos.

Se quiser entender mais de como essa wiki funciona, descrevo tudo aqui

Compartilhando

Lugares na internet onde compartilho as coisas que faço/consumo/tenho.

O que Consumo

Lista de coisas que eu uso para registrar o que eu consumo (normalmente conteúdo online).

Filmes

  • Letterboxd: Filmes que quero assistir e filmes que eu vi. Alguns contém reviews engraçadinhas (não costumo fazer reviews sérias). Além disso, tem algumas listas de filmes.

Música

  • Playlists: Minhas playlists que eu compartilho no Spotify.
  • Last.fm: Aqui ficam as músicas que já ouvi, o scrobble é feito automaticamente do Spotify, então músicas que escuto em outras plataformas não aparecem.

Editores de Texto

Sempre prefiro usar editores de texto para programar, por serem muito mais leves que IDEs.

Os que eu mais uso são o Visual Studio Code e o Neovim.

Visual Studio Code

O editor de texto que eu mais uso por ser fácil de extender e multiplataforma.

Eu uso várias extensões nele.

Vivo mudando da aparência dele, mas atualmente uso o Base16 Dark Monokai com a fonte JetBrains Mono.

Digital Garden Workflow

Todo o meu Digital Garden pode ser encontrado dentro de uma pasta no Github. A pasta é compasta de várias subpastas.

A Wiki é publicada na forma de um mdBook. Toda a estrutura da sidebar pode ser vista no arquivo SUMMARY.md.

As edições feitas no computador são realizadas usando o Visual Studio Code
